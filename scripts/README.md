# Scripts

Helper scripts for dev workflows, testing, and tooling.

- `serve-demo.sh` lance un serveur statique local pour `clients/web-overlay/demo.html`
- `make demo` est l'entrée recommandée (centralisation via Makefile)
- `build-jellyfin-plugin.sh` build local du plugin server Jellyfin
- `sync-jellyfin-refs.sh` extrait les DLL Jellyfin depuis un conteneur
